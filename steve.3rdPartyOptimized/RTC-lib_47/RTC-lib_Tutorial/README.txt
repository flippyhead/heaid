\f1\b\fs32 \cf0 RTC-lib Tutorial
\f0\b0\fs28 \
This tutorial contains elaborated examples for algorithmic composition in real time. It demonstrates how selected compositorial issues such as harmony, rhythm, and random can be mastered with the aid of the RTC-lib. Furthermore it contains several especially adapted and documented structure generators from Karlheinz Essl's "Lexikon-Sonate" (1993-2007) such as Triller, MeloChord, and Esprit.\
Please note that you have to install the latest version of the RTC-lib in order to use the patches!\
